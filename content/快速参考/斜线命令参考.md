# Claude Code 斜线命令参考

> 完整的斜线命令速查表

---

## 📋 管理命令

| 命令 | 功能 |
|------|------|
| `/clear` | 清除当前会话历史（保持权限） |
| `/exit` | 退出 Claude Code |
| `/rewind` | 回到之前状态，撤销最后操作 |
| `/login` | 切换或登录不同账户 |
| `/logout` | 登出当前账户 |

---

## ⚙️ 配置命令

| 命令 | 功能 |
|------|------|
| `/config` | 打开交互式配置菜单 |
| `/model` | 查看或切换当前模型 (haiku/sonnet/opus) |
| `/permissions` | 查看和管理权限设置 |
| `/vim` | 启用/禁用 Vim 模式 |
| `/output-style` | 改变输出格式 (compact/detailed/minimal) |

---

## 📊 分析和信息命令

| 命令 | 功能 |
|------|------|
| `/context` | 显示当前加载的上下文（文件、记忆、MCP等） |
| `/status` | 显示账户信息和当前模型 |
| `/cost` | 显示当前会话成本估计 |
| `/usage` | 显示 API 使用统计 |
| `/doctor` | 诊断 Claude Code 设置和连接 |

---

## 🔄 工作流命令

| 命令 | 功能 |
|------|------|
| `/compact` | 压缩会话以节省 token |
| `/export` | 导出当前会话（Markdown 或 JSON） |
| `/review` | 启动代码审查会话 |

---

## 🔌 集成命令

| 命令 | 功能 |
|------|------|
| `/mcp` | 查看或管理 MCP 服务器 |
| `/ide` | 检查和配置 IDE 集成 |
| `/plugin` | 列出和管理已安装插件 |
| `/hooks` | 查看和管理 Hooks |
| `/agents` | 创建、编辑或删除自定义 Agent |
| `/memory` | 编辑项目或用户记忆文件 (CLAUDE.md) |

---

## ❓ 帮助命令

| 命令 | 功能 |
|------|------|
| `/help` | 显示所有可用命令 |
| `/terminal-setup` | 配置多行输入 |
| `/feedback` | 报告问题或提交功能建议 |

## 📝 自定义命令

### 创建自定义命令

**项目级别**（团队共享）:
```
.claude/commands/
├── optimize.md
├── test-all.md
└── deploy.md
```

**用户级别**（个人使用）:
```
~/.claude/commands/
├── code-review.md
└── security-audit.md
```

### 命令文件格式

```markdown
---
name: 命令名称
description: 命令描述
---

命令内容...
```

### 使用自定义命令

```bash
/your-custom-command
```

## 💡 使用技巧

### 查看命令帮助
```bash
/help                    # 查看所有命令
/mcp --help              # 查看特定命令帮助
```

### 快速配置
```bash
/config                  # 交互式配置
/model sonnet            # 快速切换模型
/permissions set plan    # 快速设置权限模式
```

### 成本监控
```bash
/cost                    # 查看当前会话成本
/usage                   # 查看总使用量
/compact                 # 压缩以降低成本
```

### MCP 管理
```bash
/mcp                     # 查看所有服务器
/mcp add github          # 添加服务器
/mcp test github         # 测试连接
/mcp remove github       # 删除服务器
```

### Agent 管理
```bash
/agents                  # 管理 Agent
/memory                  # 编辑 CLAUDE.md
```

---

## 🎯 常用命令组合

**开始新任务：**
```bash
/clear           # 清除历史
/context         # 检查上下文
# 开始工作...
```

**成本优化：**
```bash
/cost            # 查看成本
/compact         # 压缩会话
/model haiku     # 切换到便宜模型
```

**审查和导出：**
```bash
/review          # 审查代码
/export          # 导出会话
```

**提示**: 输入 `/` 后按 Tab 可以看到自动补全建议！
