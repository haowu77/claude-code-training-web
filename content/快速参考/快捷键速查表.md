# Claude Code 快捷键速查表

> 快速参考卡片 - 打印使用

---

## 🎹 交互模式快捷键

| 快捷键 | 功能 |
|-------|------|
| `Tab` | 切换 Extended Thinking（深度思考模式） |
| `Shift+Tab` | 切换权限模式（Default ↔ Plan ↔ Auto-Accept） |
| `Alt+M` | 切换权限模式（同上） |
| `Ctrl+C` | 中断当前操作 |
| `Ctrl+L` | 清除屏幕（保留历史） |
| `Ctrl+O` | 切换详细输出 |
| `Ctrl+R` | 搜索命令历史 |
| `Esc` `Esc` | 回滚最后操作 |

## 🔄 权限模式循环

```
Shift+Tab 循环顺序：
Default → Plan → Auto-Accept → Default
```

**模式说明：**
- **Default**: 常规操作自动，敏感操作确认
- **Plan**: 所有操作显示计划，需批准
- **Auto-Accept**: 所有操作自动执行（危险！）

## 💡 使用技巧

1. **Tab 补全**: 输入 `@` 后按 Tab 显示文件建议
2. **历史搜索**: `Ctrl+R` 搜索之前的命令
3. **快速中断**: 两次 `Esc` 快速回滚
4. **批量引用**: `@src/**/*.ts` 引用多个文件
